# (Pulso Propiedades) — Frontend

**Proyecto:** Pulso Propiedades Intranet Corredores  
**Versión:** 2.0.1  
**Homepage:** https://pulsopropiedades.app/

Aplicación frontend construida con **React 18 + Vite + TypeScript**, con integración de **Redux Toolkit**, **RTK Query**, **TailwindCSS**, **i18next**, y varias librerías UI/UX (drag & drop, mapas, calendarios, editores rich text, gráficos, etc.).

---

## Stack principal

- **Runtime:** Node.js (recomendado LTS)
- **Bundler:** Vite
- **UI:** React 18
- **Lenguaje:** TypeScript
- **Estado:** Redux Toolkit + React Redux + Redux Persist
- **Estilos:** TailwindCSS + PostCSS
- **i18n:** i18next + react-i18next
- **Mapas:** mapbox-gl + react-map-gl
- **Gráficos:** apexcharts + react-apexcharts
- **D&D:** @dnd-kit + react-beautiful-dnd
- **Calendario:** FullCalendar
- **Editores:** draft-js / react-draft-wysiwyg / react-quill
- **Realtime:** socket.io-client
- **Mock API (opcional):** miragejs

---

## Requisitos

- **Node.js:** recomendado **LTS** (idealmente 18+)
- **pnpm:** recomendado como package manager del proyecto

> Nota: si tu entorno usa `nvm`, instala la versión LTS y asegúrate de usarla dentro del repo.

---

## Instalación

<!-- Instalar dependencias -->
<!-- pnpm install -->

<!-- Levantar el entorno de desarrollo (Vite) -->
<!-- pnpm start -->

<!-- Generar build optimizado -->
<!-- pnpm run build -->

<!-- Servir el build localmente -->
<!-- pnpm run preview -->
